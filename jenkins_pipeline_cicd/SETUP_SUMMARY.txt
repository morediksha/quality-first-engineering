1. Create Pipeline Job in Jenkins:

   • Name: squad-playwright-tests
   • Type: Pipeline
   • Configure → Pipeline → Pipeline script from SCM
   • SCM: Git
   • Repository URL: your-git-url
   • Script Path: Jenkinsfile

2. Job Accepts These Parameters:
   • BRANCH (default: main)
     → Git branch to test
   • TESTS_PATH (default: tests/smoke)
     → tests/smoke, tests/regression, or tests
   • SQUAD_URL_FILE (default: squad_urls.txt)
     → Your URLs file
   • HEADLESS_MODE (default: true)
     → Headless browser mode

3. Build with Parameters:
   • Click "Build with Parameters"
   • Configure each parameter
   • Click "Build"
   • Monitor console output

4. View Reports:
   • After build completes
   • Click "Allure Report" link
   • Or check Jenkins artifacts

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Problem: Python command not found
Solution: Use python3 explicitly (macOS/Linux)
         Or install Python 3.7+

Problem: pytest not found
Solution: Activate virtual environment
         source venv/bin/activate

Problem: squad_urls.txt not found
Solution: Create it: touch squad_urls.txt
         Add URLs (one per line)

Problem: Playwright not installed
Solution: playwright install chromium

Problem: Tests timeout
Solution: Increase timeout in pytest.ini
         Or check if URLs are accessible

Problem: Jenkins build fails
Solution: Check console output for errors
         Verify Git credentials
         Check Python on agent

For more help, see: README.md troubleshooting section
